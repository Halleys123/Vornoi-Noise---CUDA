# Requirements

1. CUDA Toolkit
2. Windows 95 or Later

## Usage

To compile the project, run the following command in your terminal:

```terminal
make
```

Running the compiled program can be done with:

```terminal
./build.exe
```

## Controls

- [**X**, **Y**] (Case Insensitive): Selects direction in which divisions of vornoi to change
- [SPACE]: Deselects current selection
- [**+**, **-**]: Increases/Decreases number of divisions in selected direction
